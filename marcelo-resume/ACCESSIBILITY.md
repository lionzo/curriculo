# Accessibility Notes

This static resume keeps the original template's accessibility-oriented structure:

- Semantic landmarks: `header`, `aside`, `main`, `section`, `article`, and `footer`.
- A skip link targets the main content.
- Interactive controls have accessible labels.
- Decorative SVG icons are hidden from assistive technologies with `aria-hidden="true"`.
- Links use visible text and underlines.
- Resume content is structured with headings, lists, and definition lists.
- Print styles are included in `styles.css`.

Before publishing a final production version, verify the page with:

- Keyboard navigation.
- A screen reader.
- Browser accessibility tooling.
- Contrast checks for any changed colors or assets.
